## 关键技术决策

### v1.1.1

- **1 runner 1 进程**: 简化并发模型, 用 state 文件作为 IPC
- **子进程 + log 捕获**: 用 `subprocess.Popen(stdout=PIPE)` + 后台 thread 转发到 log
- **PIPELINE marker**: 在 stdout 打 `[PIPELINE] book=X ch=Y stage=Z status=W`, dashboard 解析
- **隐式 stage context**: `llm.set_stage_context(stage, ch)`, 后续 complete/call 自动带
- **SSE 优于 WS**: 单向流足够, 浏览器原生 EventSource 简单
