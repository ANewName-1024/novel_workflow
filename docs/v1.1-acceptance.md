## 已知问题 & 后续

### v1.1 不在范围内 (M7+ 候选)
- [ ] **取消后恢复**: 现在 cancel 后不能再续跑, 未来支持断点续传
- [ ] **多本书并发**: 现在 1 runner 只跟踪 1 本, 多本书要 queue
- [ ] **WebSocket 替代 SSE**: SSE 单向, 双向 (如客户端反向控制) 要 WS
- [ ] **鉴权细粒度**: 现在全局 Basic Auth, 未来加 per-book RBAC
- [ ] **历史章节 dry-run**: 在写之前 LLM 生成大纲预览, 人工确认再真写

### 待观察 (实战后)
- [ ] 大章节 (>5000 字) 写时 dashboard SSE 是否断流
- [ ] 7 天 metrics 聚合性能 (JSONL 行数 >1000 时)
- [ ] 多用户同时看 dashboard 时的轮询风暴
